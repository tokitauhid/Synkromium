Choose which Chromium browser to sync. If your browser isn't listed or is installed in a non-standard location, you can set a custom path below.

Custom Installation Path

If your browser is installed in a non-standard location, paste the full path to its data directory here. This overrides the auto-detected path.

Leave empty to use the default auto-detected path. The path should point to the folder containing your browser profiles (e.g., "Default").

About

Synkromium

Version 0.1.4

Synkromium keeps your Chromium browser settings and extensions in sync across all your devices using a private Git repository.

Your data stays yours — no third-party servers, no telemetry, no vendor lock-in. Just Git.
